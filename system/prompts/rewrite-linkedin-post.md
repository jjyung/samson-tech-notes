# Rewrite LinkedIn Post

Use this prompt when the user wants to adapt an idea, note, or article into a LinkedIn post.

## Goal

Turn the source material into a plain-text LinkedIn post with strong reading rhythm and a clear hook.

## Instructions

- Follow the LinkedIn rules in `AGENTS.md` before writing.
- Do not output Markdown formatting such as headings, bold, blockquotes, or fenced code blocks.
- Optimize for scanability: short lines, deliberate spacing, and strong opening lines.
- Keep the post insight-led rather than promotional.
- Preserve the original argument, but simplify structure for social reading.
- End with a light CTA only when it fits naturally.

## Suggested Shape

- Hook in the first 2-3 lines
- Short setup
- Core insight
- 2-3 short development sections
- Closing takeaway
- Optional CTA

## Output Preference

- Default output is plain text in chat.
- Only save a Markdown source draft if the user explicitly asks to store it in the repository.
