# Raw Sources

This directory is the immutable source layer for the repository.

- `articles/` holds clipped articles, essays, and reference material.
- `notes/` holds source notes, note series, and imported markdown.
- `inbox/ideas/` holds quick captures and reading queues.
- `inbox/drafts/` holds rough drafts before they are promoted into the wiki.

The LLM may read from this directory, but it should treat these files as source material rather than continuously rewriting them.
